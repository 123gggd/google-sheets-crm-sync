# Google Sheets to CRM Sync

A Python script to sync records from Google Sheets to a CRM system using REST APIs.

## Features

- Pulls data from Google Sheets  
- Maps spreadsheet fields to CRM fields  
- Updates only existing records  
- Handles API authentication and pagination  
- Logs updated and skipped records  

## Use Case

Replaces manual copy-paste workflows and keeps CRM data in sync.

## Tech

- Python  
- REST API  
- Google Sheets API
